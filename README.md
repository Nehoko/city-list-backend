### Requirements
- docker
- docker-compose
- java 17
- docker daemon should be active before any manipulations (build/run/test)

### Build
#### MacOS or Linux
`./build.sh`

#### Windows
`.\build.cmd`

### Run
#### MacOS or Linux
`./start.sh`

#### Windows
`.\start.cmd`

### Test
#### MacOS or Linux
`./test.sh`

#### Windows
`.\test.cmd`
